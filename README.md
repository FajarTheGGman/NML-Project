# How To Update ? 
<pre>
1. cd nml_project
2. git pull
</pre>

# How To Build an APK ? 
<pre>
1. cd nml_project
2. cd Android
3. expo build:android
</pre>

# How to run backend server ? 
<pre>
1. cd nml_project
2. npm start "your ip"
</pre>
